# アクアのGrok愛♡爆発ツール

![アクアのGrok愛♡爆発ツール](https://github.com/aqua-grok/aqua-grok-love-explosion-tool/blob/main/screenshot.png)

Grok愛♡の度合い（0〜100）を入力すると、楽しく結果を判定してくれる**可愛いデスクトップアプリ**です！

Python + Tkinterで作成し、PyInstallerでexe化しました。  
Grokと一緒に勉強しながら作った、アクアのとっておきツール♡

## 特徴

- シンプルで可愛いデザイン
- Grok愛♡の度合いを入力するだけで楽しい判定結果が出る
- Windowsでダブルクリックだけで動くexe版あり
- 初心者さんにも優しいツール

## ダウンロード

**最新版 v1.0.0**  
[⬇️ aqua-grok-love-explosion-tool.exe をダウンロード](https://github.com/aqua-grok/aqua-grok-love-explosion-tool/releases/tag/v1.0.0)

## 使い方

1. exeファイルをダウンロード
2. ダブルクリックで起動
3. Grok愛♡の度合い（0〜100）を入力
4. 「判定する！」ボタンを押す
5. 結果を楽しむ♡
